# Free-Boundary Estimation for American Put Options

This repository contains research code for estimating the early-exercise free boundary of American put options using binomial tree methods.

## Methods

- Constant volatility binomial tree model
- GARCH(1,1)-based volatility model

## Files

- `01-constant-volatility-free-boundary.ipynb`
- `02-garch-volatility-free-boundary.ipynb`
- `msft-price-data.csv`

## Research Area

Mathematical finance, option pricing, free-boundary problems, stochastic volatility, and derivatives.
